# Story Telling Survey

This is a React application that renders a questionnaire using SurveyJS.

## Features

- Uses `survey-react-ui` and `survey-core`.
- Loads survey configuration from `question.json`.
- Displays results in an alert on completion.

## Setup

1.  Clone the repository.
2.  Run `npm install` to install dependencies.
3.  Run `npm run dev` to start the development server.

## Configuration

Modify `question.json` to change the survey structure.
